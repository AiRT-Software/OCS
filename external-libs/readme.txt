Please, download the following libraries in order to compile the project. They are required by the "install-deps-up2.sh" script.
We provide the specific versions we have tested. Newer versions may or may not work.

flann-1.9.1
eigen-3.3.4
librealsense-2.11.0
zeromq-4.2.1

0mq 'highlevel' C++ bindings. Version at August 8th, 2017
You can download it from:
https://github.com/zeromq/zmqpp/archive/581ba8b6b0e0080d419b6424575c06fd77de2451.zip
It should be in a directory called zmqpp

rapidjson. A fast JSON parser/generator for C++ with both SAX/DOM style API. Version at January 19th, 2018
https://github.com/Tencent/rapidjson/archive/fc7cda78a9b25e986f245ed0655df0aa4b71bc3b.zip
It should be in a directory called rapidjson

glm. OpenGL Mathematics. Version at April 20th, 2018
https://github.com/g-truc/glm/archive/347e767ac46006ad168363a9eef20b9e1e2cde0d.zip
It should be in a directory called glm-master
